Passing data from Parent to Child.

Use @Input() decorator in the Child component.

This variable should contain, for example, text which will be passed from Parent.

In the square brackets [variable]="value" in Parent component pass data which you need to share.
Passing data from Child to Parent.

Use @Output and EventEmitter in the Child component.

Code example: @Output() messageEvent = new EventEmitter().

Create function which will send your message to the Parent component.

Code example: this.messageEvent.emit(this.YOUR_MESSAGE).

In Parent component inside child tag use (messageEvent)="recieveMessage($event)".

Do not forget to create function which will take and use data in the way you need.
From the Child to Parent

We will use @ViewChild() to pass data in this case.

Code example: @ViewChild(ChildViewCComponent) child.

Use the correct lifecycle hook to modify the data.
